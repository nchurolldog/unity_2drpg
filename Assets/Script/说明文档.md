# 弓箭射击方向说明

## 整体数据流

```
移动输入(WASD/摇杆)
       │
       ▼
┌─────────────────────┐
│  PlayerController   │
│  _moveinput = 方向   │
└────────┬────────────┘
         │
         │  Player_Bow 每帧读取 player._moveinput
         ▼
┌─────────────────────┐
│    Player_Bow       │
│  HandleAiming()     │
│  aimDirection = 方向 │──────► Animator (aimx, aimy 参数)
└────────┬────────────┘
         │
         │  攻击输入触发
         ▼
┌─────────────────────┐
│  PlayerController   │
│  _currentAttack     │
│  Provider.Attack()  │
└────────┬────────────┘
         │
         ▼
┌─────────────────────┐
│  Player_Bow.Attack()│
│  isShooting = true  │──────► 播放射击动画
└────────┬────────────┘
         │
         │  动画事件回调
         ▼
┌─────────────────────┐
│  Player_Bow.Shoot() │
│  用 aimDirection    │──────► Arrow._derection = aimDirection
│  生成箭矢           │
└─────────────────────┘
```

## 关键变量追踪

### aimDirection 何时更新？

**每帧**在 `Player_Bow.HandleAiming()`（`Update` 中调用）：

```csharp
// Player_Bow.cs 第45-53行
private void HandleAiming()
{
    Vector2 moveinput = player._moveinput;  // ← 从 PlayerController 读移动方向
    if (moveinput.x != 0 || moveinput.y != 0)
    {
        aimDirection = new Vector2(moveinput.x, moveinput.y).normalized;
        animator.SetFloat("aimx", aimDirection.x);
        animator.SetFloat("amiy", aimDirection.y);
    }
}
```

**重要**：当玩家松开移动键（`_moveinput` 归零）时，`aimDirection` **保持不变**，保留最后一次移动方向。所以站着不动时射击方向是最后面向的方向。

### aimDirection 初始值？

```csharp
Vector2 aimDirection = Vector2.right;  // 默认向右 (1, 0)
```

### 箭矢方向怎么传给 Arrow？

```csharp
// Player_Bow.cs 第55-61行（动画事件调用）
void Shoot()
{
    if (timer <= 0)
    {
        Arrow arrow = Instantiate(arrowPrefab, launchPoint.position, Quaternion.identity)
            .GetComponent<Arrow>();
        arrow._derection = aimDirection;  // ← 方向在这里赋值
        timer = shootCoolDown;
    }
    animator.SetBool("isShooting", false);
}
```

## 完整射击流程（一次按键）

| 步骤 | 发生了什么 | 哪个脚本 |
|------|-----------|---------|
| 1 | 玩家按移动键 | PlayerController 更新 `_moveinput` |
| 2 | 每帧 Update | Player_Bow.HandleAiming() 同步 `aimDirection` |
| 3 | 玩家按攻击键 | PlayerController → `_currentAttackProvider.Attack()` |
| 4 | Attack() 被调用 | Player_Bow 设 `isShooting = true` |
| 5 | Animator 检测到 isShooting | 播放射击动画 |
| 6 | 动画事件触发 Shoot() | 用 `aimDirection` 生成箭矢 |
| 7 | Shoot() 结束 | `isShooting = false`，进入冷却 |

## 切换武器后的行为

`PlayerChangEquipment.Toggle()` 切换时：

```
弓箭手模式：
  bow.enabled = true
  meleeAttack.enabled = false
  _currentAttackProvider = bow → 攻击键触发 Player_Bow.Attack()

骑士模式：
  bow.enabled = false
  meleeAttack.enabled = true
  _currentAttackProvider = meleeAttack → 攻击键触发 MeleeAttack.Attack()
```

## 注意事项

- **瞄准方向来源于移动输入**：`aimDirection` 跟随最后按下的移动方向。这是用户要求保留的设计（未改动第5点）。
- **静止时方向保持**：松手后 `aimDirection` 不会归零，保留最后朝向。
- **近战攻击方向**：由 `transform.localScale` 控制翻转（`PlayerController.cs:58`），根据 `_moveinput.x` 的正负决定角色朝向。
